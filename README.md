# AI Rhyme Bot

**AI Rhyme Bot** — это умный телеграм-бот, который помогает подбирать рифмы не только к отдельным словам, но и к целым фразам. В отличие от обычных генераторов, он использует **OpenAI (ChatGPT)**, чтобы сохранять ритм, смысл и юмор в ответах.

## Что умеет бот
* **Рифмы к словам:** "Мышка" → "Пышка, крышка".
* **Рифмы к фразам:** "Красная мышка" → "Ужасная стрижка, опасная вышка".
* **Адаптивность:** Понимает контекст и старается генерировать смешные или остроумные варианты.
* **История:** Сохраняет запросы пользователей в базу данных (MongoDB).

## Стек технологий
* **Core:** Node.js (JavaScript)
* **Telegram:** node-telegram-bot-api
* **AI:** OpenAI API (GPT-3.5 Turbo)
* **Database:** MongoDB + Mongoose
* **Tools:** Dotenv, Git

Перед запуском убедитесь, что у вас установлены:
* [Node.js](https://nodejs.org/) (версия 18 или выше)
* [MongoDB](https://www.mongodb.com/try/download/community) (для сохранения истории запросов)
* [Git](https://git-scm.com/)

Также вам понадобятся:
1.  **API Key** от [OpenAI](https://platform.openai.com/api-keys).
2.  **Токен бота**, полученный у [@BotFather](https://t.me/BotFather) в Telegram.
